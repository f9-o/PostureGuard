## 🌟 الرؤية (The Vision)
آلام الظهر، إجهاد العين، وتدني الإنتاجية هي أعداء الموظف المكتبي الحديث. **PostureGuard+** ليس مجرد نظام مراقبة صارم؛ بل هو رفيق ذكي يهتم بسلامتك المهنية مع احترام مطلق لخصوصيتك. يعتمد النظام على نهج "المراقبة المتدرجة الذكية" التي تبدأ بصمت، وتتدخل بفاعلية حين يبلغ التعب ذروته.

## 🚀 المميزات الأساسية (Key Features)
1. ⌨️ **تحليل نمط الكتابة الصامت (Silent Productivity Analysis)**
   يحلل النظام أوقات التأخير بين النقرات وتزايد استخدام زر (Backspace) كمؤشر لا شعوري على التعب الذهني بدون الحاجة لفتح الكاميرا وبدون الاحتفاظ بما يُكتب!
2. 👁️ **الرؤية الحيوية الهندسية (Biometric Vision)**
   محرك ذكاء اصطناعي محلي معزز بنموذج (MediaPipe) يقيس الانحناءات الهندسية (المسافة بين الكتف والأذن) للتأكد من استقامة الظهر، ويحسب معدل الرمش (EAR) لمنع جفاف العين.
3. 🔒 **الخصوصية أولاً (PDPL Compliant)**
   جميع المعالجة تتم محلياً وبشكل لحظي (On-device Edge Computing). لا تُحفظ الصور أو تُرفع لأي خادم. سرية معلومات الموظف أولوية قصوى.
4. ⏸️ **التدخل الذكي (Smart Breaks & Teams Integration)**
   بمجرد وصول عداد الإرهاق للحد الأقصى، يتدخل النظام برفق؛ يحجب الشاشة مؤقتاً لتخفيف إجهاد العين، ويبدل حالة (Microsoft Teams) تلقائياً إلى "الرجاء عدم الإزعاج".

## 🛠️ البنية التقنية (Tech Stack)
* **Backend:** Python, FastAPI, Uvicorn, WebSockets.
* **Computer Vision:** OpenCV, Google MediaPipe (Pose & Face Landmarker).
* **Silent Tracking:** Pynput.
* **Frontend:** HTML5, TailwindCSS (Glassmorphism), Chart.js.

## ⚙️ طريقة التشغيل المطوّر (Installation & Setup)
هذا المشروع مصمم ليعمل بسلاسة مطلقة. اتبع الخطوات التالية لإعداده محلياً:

### 1. المتطلبات المسبقة
* تثبيت `Python 3.10` أو أحدث. (الرجاء التأكد من تفعيل خيار `Add to PATH` أثناء التثبيت).
* كاميرا ويب (Webcam) فعالة.

### 2. تجهيز البيئة
افتح موجه الأوامر (Terminal) وضع الأوامر التالية بالترتيب:

```bash
# إنشاء بيئة عمل نظيفة ومعزولة
python -m venv venv

# تفعيل البيئة (ويندوز)
.\venv\Scripts\activate
# ولتفعيل البيئة في (ماك/لينكس): source venv/bin/activate

# تثبيت متطلبات النظام والمكاتب المرفقة بالكامل
pip install -r requirements.txt
```

### 3. إطلاق المنصة
بمجرد انتهاء التثبيت، أطلق التطبيق بأعلى أداء عبر:
```bash
uvicorn main:app --reload
```
الآن، قم بزيارة الرابط التالي في المتصفح لرؤية النظام يعمل:
👉 **[http://127.0.0.1:8000](http://127.0.0.1:8000)**

## 🧪 للمحكمين (Demo Instructions)
لاختبار تدخل النظام الآلي (مرحلة الإرهاق العالي) بسرعة أثناء التقييم دون الحاجة للانتظار:
1. قم بالضغط السريع والمتكرر على زر `Backspace` أكثر من 15 مرة متتالية.
2. سيلاحظ النظام انخفاض الأداء وقفزة في معدل الأعطال.
3. شاهد كيف يستجيب الواجهة بتحويل الحالة إلى التنبيه الأحمر وتشغيل وضع "عدم الإزعاج".
